Student Performance Prediction

Predict final grades using attendance, study habits, prior marks, assignment completion, and sleep balance. This frontend mirrors the Streamlit app in a clean, professional HTML layout.

Project Overview

This web interface is designed for academic teams, schools, and student analytics demos. It includes a prediction form, dashboard charts, and insight summaries in one responsive layout.
